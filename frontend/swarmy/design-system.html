Rules
01. Black/white/grey for everything structural. Color is reserved for status only — don't use it for branding, decoration, or emphasis.
02. IBM Plex Mono everywhere. IBM Plex Sans only for long prose passages.
03. All text is uppercase + tracked when it's a label. Sentence case for everything else.
04. Border radius: 2px maximum. No pill buttons. No rounded cards.
05. Borders: 1px solid. Two weights — --700 for primary, --800 for subtle/nested.
06. No box shadows. Depth is created with background tones (--black, --900, --800).
07. One primary button per screen. Everything else is secondary or ghost.
08. Numbers are data. Make them big. Let them breathe.
09. Error states use color sparingly — error text, not error backgrounds. Avoid alarming the user.
10. When in doubt, remove. If it doesn't carry information, it shouldn't be there.
